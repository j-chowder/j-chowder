## Projects I'm Proud Of

### 📚 [Text Classification of Japanese Literature](https://github.com/j-chowder)
Took 110,000 unstructured records and turned them into 9 coherent themes using BERTopic for topic modeling. Did deep EDA across 52 variables and built classification models that went from 29% to 63% accuracy. The real win was getting inconsistent data from multiple sources into one clean, validated 15k+ record dataset first — garbage in, garbage out is real.

### 🎌 [Full-Stack Hybrid Anime Recommender](https://github.com/j-chowder)
Built a recommender system on 10M+ ratings using item-based collaborative filtering + content-based filtering. The data cleaning alone was a project — cut the anime dataset by 90% and ratings by 46% through relevance filtering and franchise deduplication. Backed it with a PostgreSQL pipeline, metadata from the Jikan API, and a FastAPI backend. Actually works.
